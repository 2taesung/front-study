# 14. {}와 Object

```typescript
const x: {} = 'hello';
const y: Object = 'hi';
```

\=> {}, Object는 모든 타입을 허용함&#x20;

\=> 아마도 js에서 모든게 object이기 때문(추측)

\=> 단, null, undefined 제외

하지만 여기서&#x20;

아래 코드 첫줄처럼 unknown은 {}를 포함한다.

왜? 모든 타임을 뜻하니까.

```typescript
const z: unknown = 'hi';

// unknown = {} | null | undefined
if (z) {
    z; // {} 이다. 왜 ? null, undefined가 떨어져 나가니까.
} else {
    z; // 근데 왜 여기는 여전히 unknown이지 ????(질문중)
}

```
